API "Lista de tarefas"

Dica: use o POSTMAN ou INSOMNIA para testar a API e verificar as respostas!

Adicionando uma tarefa:

  • URL da requisição: http://phpapi.localhost/api/
  • Método da requisição: POST
  • Dados da requisição (BODY):
    • date → string
    • description → string
    • priority → string

Obtendo uma tarefa:

  • URL da requisição: http://phpapi.localhost/api/num/
    • num → integer é o ID da tarefa.
  • Método da requisição: GET

Obtendo a lista de tarefas:

  • Método da requisição: GET
  • URL da requisição: http://phpapi.localhost/api/
    • por default, ordena pela data (date) crescente (ASC).
     
  • URL da requisição: http://phpapi.localhost/api/field/
    • field → string é o campo usado para ordenar ('task', 'date', 'description' ou 'priority')
     
  • URL da requisição: http://phpapi.localhost/api/field/direction/
    • field → string é o campo usado para ordenar ('task', 'date', 'description' ou 'priority')
    direction → string é a direção do ordenamento ('asc' ou 'desc')

Editando uma tarefa:

  • URL da requisição: http://phpapi.localhost/api/
  • Método da requisição: PUT
  • Dados da requisição (BODY):
    • id → integer (obrigatório)
    • date → string
    • description → string
    • priority → string
    Não é necessário alterar / enviar todos os campos.

Apagando uma tarefa:

  • Requisição: http://phpapi.localhost/api/num/
    • num → integer é o id da tarefa.
  • Método da requisição: DELETE
